# Customer-Churn-Prediction
🚀 Excited to share my latest Data Science project: **Customer Churn Prediction using Machine Learning**!

As part of my journey toward becoming a Data Scientist, I built an end-to-end machine learning project to predict whether a customer is likely to leave a telecom company based on their demographic and service information.

### 🔹 What I worked on:

* Cleaned and preprocessed real-world customer data
* Performed Exploratory Data Analysis (EDA)
* Encoded categorical features and handled missing values
* Built a Random Forest Classification model
* Evaluated the model using Accuracy, Precision, Recall, F1-Score, and Confusion Matrix
* Saved the trained model using Joblib
* Developed a Streamlit web application for real-time predictions
* Managed the project using Git and GitHub

### 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Streamlit
* Git & GitHub

This project helped me understand the complete machine learning workflow from data preprocessing to model deployment.
